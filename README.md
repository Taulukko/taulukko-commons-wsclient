Taulukko Rest Client
========================

To use Tauluko Maven Mirror, add in your settings.xml

	<mirror>
			<id>Central Internal</id>
			<url>http://repository.taulukko.com:8080/repository/internal</url>
			<mirrorOf>central</mirrorOf>
	</mirror>
